---
title: useScriptLeaflet
---

Use `useScriptLeaflet()`{lang="ts"} when you need direct access to the Leaflet namespace or are building your own map component.

::script-types{script-key="leaflet" filter="useScriptLeaflet"}
::

```ts
const { onLoaded } = useScriptLeaflet()

onLoaded(({ L }) => {
  const bounds = L.latLngBounds(points)
  // Use the Leaflet API directly.
})
```
