---
title: <ScriptMapLibreMarker>
---

Adds a reactive MapLibre `Marker`. Supply a unique `aria-label` so its generated DOM element has a useful accessible name.

::script-types{script-key="maplibre" filter="ScriptMapLibreMarker"}
::

Nest a [`<ScriptMapLibrePopup>`{lang="html"}](/scripts/maplibre/api/popup) in the default slot to bind it to the marker. Use `options.draggable` and the drag events for an editable marker.
