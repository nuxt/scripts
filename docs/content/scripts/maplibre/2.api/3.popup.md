---
title: <ScriptMapLibrePopup>
---

Renders slotted HTML in a MapLibre `Popup`. Nest it inside a marker, or pass `position` for a standalone popup. Use the reactive `open` prop to control visibility.

`new maplibregl.Popup()`{lang="ts"} reads the `options` object at creation time. Options backed by MapLibre public setters also update reactively: `className`, `maxWidth`, `offset`, `padding`, and `subpixelPositioning`. Other options are initialization-only.

::
