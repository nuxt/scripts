A quick walkthrough to get a Google Map with markers running in your Nuxt app.

## Minimal Map

```vue
<template>
  <ScriptGoogleMaps
    api-key="YOUR_API_KEY"
    :center="{ lat: -33.8688, lng: 151.2093 }"
    :zoom="12"
  />
</template>
```

This renders a static map placeholder that loads the interactive map on hover. See [Setup](/scripts/google-maps#setup) for API key configuration via environment variables.

## Adding Markers

Use `ScriptGoogleMapsAdvancedMarkerElement` to place markers on the map. The `:position` prop sets where the marker appears.

```vue
<template>
  <ScriptGoogleMaps
    api-key="YOUR_API_KEY"
    :center="{ lat: -33.8688, lng: 151.2093 }"
    :zoom="12"
  >
    <ScriptGoogleMapsAdvancedMarkerElement
      :position="{ lat: -33.8688, lng: 151.2093 }"
    />

    <ScriptGoogleMapsAdvancedMarkerElement
      :position="{ lat: -33.8750, lng: 151.2150 }"
    />
  </ScriptGoogleMaps>
</template>
```

## Info Windows

Nest a `ScriptGoogleMapsInfoWindow` inside a marker. It opens automatically when the marker is clicked.

```vue
<template>
  <ScriptGoogleMaps
    api-key="YOUR_API_KEY"
    :center="{ lat: -33.8688, lng: 151.2093 }"
    :zoom="12"
  >
    <ScriptGoogleMapsAdvancedMarkerElement
      :position="{ lat: -33.8688, lng: 151.2093 }"
    >
      <ScriptGoogleMapsInfoWindow>
        <h3>Sydney Opera House</h3>
        <p>Iconic performing arts venue</p>
      </ScriptGoogleMapsInfoWindow>
    </ScriptGoogleMapsAdvancedMarkerElement>
  </ScriptGoogleMaps>
</template>
```

## Custom Marker Content

The `#content` slot replaces the default pin with any HTML or Vue template.

```vue
<template>
  <ScriptGoogleMaps
    api-key="YOUR_API_KEY"
    :center="{ lat: -33.8688, lng: 151.2093 }"
    :zoom="12"
  >
    <ScriptGoogleMapsAdvancedMarkerElement
      :position="{ lat: -33.8688, lng: 151.2093 }"
    >
      <template #content>
        <div style="background: #1a73e8; color: white; padding: 4px 10px; border-radius: 16px; font-weight: bold;">
          $1.2M
        </div>
      </template>
    </ScriptGoogleMapsAdvancedMarkerElement>
  </ScriptGoogleMaps>
</template>
```

## Custom Popups

For full control over popup styling (no InfoWindow constraints), use `ScriptGoogleMapsOverlayView` with `v-model:open`.

```vue
<script setup lang="ts">
const showPopup = ref(false)
</script>

<template>
  <ScriptGoogleMaps
    api-key="YOUR_API_KEY"
    :center="{ lat: -33.8688, lng: 151.2093 }"
    :zoom="12"
  >
    <ScriptGoogleMapsAdvancedMarkerElement
      :position="{ lat: -33.8688, lng: 151.2093 }"
      @click="showPopup = !showPopup"
    >
      <ScriptGoogleMapsOverlayView
        v-model:open="showPopup"
        anchor="bottom-center"
        :offset="{ x: 0, y: -50 }"
      >
        <div style="background: white; padding: 12px; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.3);">
          <strong>Custom Popup</strong>
          <p>Any Vue content, fully styled</p>
          <button @click.stop="showPopup = false">
            Close
          </button>
        </div>
      </ScriptGoogleMapsOverlayView>
    </ScriptGoogleMapsAdvancedMarkerElement>
  </ScriptGoogleMaps>
</template>
```

## Dynamic Markers

Render markers from data with `v-for`.

```vue
<script setup lang="ts">
const places = ref([
  { id: 1, name: 'Opera House', position: { lat: -33.8568, lng: 151.2153 } },
  { id: 2, name: 'Harbour Bridge', position: { lat: -33.8523, lng: 151.2108 } },
  { id: 3, name: 'Bondi Beach', position: { lat: -33.8908, lng: 151.2743 } },
])
</script>

<template>
  <ScriptGoogleMaps
    api-key="YOUR_API_KEY"
    :center="{ lat: -33.8688, lng: 151.2093 }"
    :zoom="11"
  >
    <ScriptGoogleMapsAdvancedMarkerElement
      v-for="place in places"
      :key="place.id"
      :position="place.position"
    >
      <ScriptGoogleMapsInfoWindow>
        <strong>{{ place.name }}</strong>
      </ScriptGoogleMapsInfoWindow>
    </ScriptGoogleMapsAdvancedMarkerElement>
  </ScriptGoogleMaps>
</template>
```
