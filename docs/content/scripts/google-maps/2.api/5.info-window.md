---
title: <ScriptGoogleMapsInfoWindow>
---

Information window that automatically opens on parent marker click. Place as a child of [`<ScriptGoogleMapsMarker>`{lang="html"}](/scripts/google-maps/api/marker) or [`<ScriptGoogleMapsAdvancedMarkerElement>`{lang="html"}](/scripts/google-maps/api/advanced-marker-element).

::script-types{script-key="google-maps" filter="ScriptGoogleMapsInfoWindow"}
::

## Usage

```vue
<template>
  <ScriptGoogleMaps api-key="your-api-key">
    <ScriptGoogleMapsAdvancedMarkerElement
      :position="{ lat: -34.397, lng: 150.644 }"
    >
      <ScriptGoogleMapsInfoWindow>
        <div>
          <h3>Sydney, Australia</h3>
          <p>A great city!</p>
        </div>
      </ScriptGoogleMapsInfoWindow>
    </ScriptGoogleMapsAdvancedMarkerElement>
  </ScriptGoogleMaps>
</template>
```
