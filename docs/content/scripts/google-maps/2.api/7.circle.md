---
title: <ScriptGoogleMapsCircle>
---

Circular overlay on the map. Place inside a [`<ScriptGoogleMaps>`{lang="html"}](/scripts/google-maps/api/script-google-maps) component.

::script-types{script-key="google-maps" filter="ScriptGoogleMapsCircle"}
::

## Usage

```vue
<template>
  <ScriptGoogleMaps api-key="your-api-key">
    <ScriptGoogleMapsCircle
      :options="{
        center: { lat: -34.397, lng: 150.644 },
        radius: 1000,
        fillColor: '#FF0000',
        fillOpacity: 0.35,
        strokeColor: '#FF0000',
        strokeOpacity: 0.8,
        strokeWeight: 2,
      }"
    />
  </ScriptGoogleMaps>
</template>
```
