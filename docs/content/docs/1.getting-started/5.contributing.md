## Prepare your environment

`@nuxt/scripts` is a [pnpm](https://pnpm.io) workspace. Use pnpm to install and link its dependencies.

After cloning the repository, run:

```sh
pnpm i && pnpm dev:prepare
```

## Run the playground

Start the playground development server with:

```sh
pnpm dev
```

## Check your changes

Run the repository's lint and test commands before submitting a change:

```sh
pnpm lint
pnpm test:run
```
